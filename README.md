* 판매 분석을 위한 매출차트
* 스프링 시큐리티 기반의 OAuth2 로그인
* 상품 등록, 재고 관리
* 주문 취소와 배송 관리
* 포인트 적립
* 결제하기

* Java 8 ➡️  Java 17
* Spring Framework 5 ➡️  Spring Boot 3.1.6
* Maven ➡️  Gradle
* Mybatis ➡️  JPA
